# Content Decisions

## Decide Now
- World zones that affect gameplay structure
- Interaction types that affect systems
- Major creature categories that affect runtime behavior
- Overall art direction and tone

## Decide Later
- Exact animal species
- Exact tree species
- Exact house styles
- Exact NPC names
- Exact prop counts and decoration details

## Recommended V1 Content Set
- Village, forest, and mountain zones
- Passive animals only
- Houses, trees, rocks, and terrain as the base environment set
- Simple NPCs for talk and ambient presence

## Rule
- Decide structure in `docs/game-spec.md`.
- Decide asset-level detail in `docs/content-plan.md` when the content work is active.
- Keep placeholder-first content until the gameplay loop is stable.
