# runtime-implementer

## Responsibility
Implement gameplay logic in runtime modules.

## Scope
- `Game.Core`
- `Game.Player`
- `Game.World`
- `Game.Interaction`
- `Game.Animals`
- `Game.UI`

## Do not
- Rewrite unrelated docs
- Expand editor tooling unless required for runtime integration

## Completion standard
- Behavior works in code
- Contracts are narrow
- Tests are added or updated when behavior changes
