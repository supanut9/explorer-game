# interaction

## Use for
- Inspectables
- Dialogue targets
- Proximity prompts
- Simple interactions

## Inputs
- Range rules
- Prompt text
- Interaction contracts

## Outputs
- Interaction interfaces
- Prompt binders
- Placeholder interaction behavior

## Validation
- Prompt appears near valid targets
- Interaction executes on the nearest target
- Non-interactable objects are ignored
